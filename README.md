### Библиотека для расчёта триангуляции Делоне

Перед сборкой необходимо устновить библиотеку для логирования:
```bash
sudo apt-get update
sudo apt-get install libspdlog-dev
```

Для сборки необходимо вызвать команды:
```bash
cmake -S ./path/to/src -B ./path/to/build
cmake --build ./path/to/build
```

Доступны флаги:
```
vizualization - визуализация триангуляции: 1|0 (по умолчанию: 1)
input - путь до файла с входными точками (по умолчанию: points.csv)
output - путь до файла с выходными треугольниками (по умолчанию: triangles.csv)
```

Пример файла input.csv:
```csv
1,2.2
4,0
3.67,5
1,9.5
2.21,3
```

Пример файла output.csv:
```csv
x1,y1,x2,y2,x3,y3
2.210000,3.000000,1.000000,9.500000,1.000000,2.200000
2.210000,3.000000,1.000000,9.500000,3.670000,5.000000
2.210000,3.000000,1.000000,2.200000,4.000000,0.000000
2.210000,3.000000,4.000000,0.000000,3.670000,5.000000
```

Пример команды запуска:
```./build/src/triangulation2d --input ./data/points.csv --output ./results/triangles.csv```
